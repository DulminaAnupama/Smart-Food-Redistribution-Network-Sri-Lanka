# 🍽️ Food Redistribution & Security Analysis

## 📌 Project Overview
This project analyzes food availability, GDP, population, and urbanization to identify food security patterns.

## 🚀 Features
- Data cleaning & preprocessing
- Dashboard visualization (Power BI / Streamlit)
- AI-based insights

## 🛠️ Technologies Used
- Python (Pandas, NumPy, Matplotlib)
- R (optional)
- Power BI
- Streamlit

## 📊 Data Sources
- FAOSTAT
- Sri Lanka GDP & Population datasets
